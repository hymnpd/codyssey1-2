[미션 1] (기획) :한글의 우수성과 아름다움을 보여주는 전통적인 브랜드(브랜드 정체성 수립)
목표: 한글의 우수성과 아름다움을 보여주는 전통적인 브랜드의 이미지 현대적으로 세련되게 표현한다. 
수행 내용:
가상 브랜드 선정하기 : 이리얏
브랜드 아이덴티티 정의: 
타겟- 한글과 관련된 박물관을 찾는 외국인
톤앤매너(분위기): 전통적인 이미지와 현대적 이미지가 고르게 섞인 디자인
USP(차별점): 빠른 전개와 우리나라 전통적인 디자인을 살리는 조화가 돋보이도록 한다.
핵심 메시지: 한글의 디자인과 우수성을 발견하라.
결과물: 스토리보드 문서의 '브랜드 정의' 섹션 작성.
