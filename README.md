codyssey1-2
🚀 AI 브랜드 광고 제작 미션 로드맵
[미션 1] 브랜드 정체성 수립 (기획 단계)
목표: 광고할 브랜드의 뼈대를 잡습니다.
수행 내용:
가상 또는 실제 브랜드 1개 선정.
브랜드 아이덴티티 정의: 타겟, 톤앤매너(분위기), USP(차별점) 설정.
핵심 메시지: 광고를 통해 전달할 단 하나의 문장 결정.
결과물: 스토리보드 문서의 '브랜드 정의' 섹션 작성.
[미션 2] 스토리보드 설계 및 도구 매칭 (설계 단계)
목표: 10초 내외의 영상을 씬(Scene) 단위로 구체화합니다.
수행 내용:
전체 영상을 2~4개의 씬으로 나눕니다.
씬별 필수 필드 작성: 길이, 목표 메시지, 화면 구성, 카피 작성.
도구 선정: 이미지(Midjourney 등), 비디오(Runway 등), 오디오(Suno 등) 도구를 왜 선택했는지 이유 기술.
결과물: 스토리보드 초안 완성.
[미션 3] 멀티모달 소스 생성 및 프롬프트 엔지니어링 (제작 단계)
목표: AI 도구를 활용해 고품질의 이미지, 영상, 오디오 소스를 추출합니다.
수행 내용:
이미지 생성 후 비디오로 변환하거나 직접 비디오 생성.
배경음악(BGM) 또는 내레이션 생성.
프롬프트 수정 기록: 최소 1개 씬에서 "수정 전/후" 비교 및 이유 기록 (예: "인물 디테일이 너무 과해 실루엣 중심으로 수정").
결과물: 개별 미디어 파일들(PNG, MP4, WAV 등) 및 프롬프트 로그.
[미션 4] 영상 통합 편집 및 브랜드 각인 (완성 단계)
목표: 생성된 소스들을 하나의 광고 영상으로 조립합니다.
수행 내용:
컷 편집 도구(CapCut, Premiere 등)를 사용하여 소스 배치.
브랜드 장치 삽입: 마지막 3~5초 구간에 로고, 슬로건, CTA(구매 유도 등) 포함.
기술 스펙 준수: 10초 이내, 720p~1080p 해상도, H.264 코덱 적용.
결과물: 최종 광고 영상 파일 (MP4).
[미션 5] 제작 파이프라인 문서화 (회고 단계)
목표: 본인이 설계한 제작 과정을 논리적으로 설명합니다.
수행 내용:
사용된 도구들의 강점과 약점 정리.
캐릭터/화풍 일관성을 유지하기 위해 사용한 전략(예: Reference 기능 사용 등) 기술.
전체 제작 파이프라인의 의사결정 이유 설명.
결과물: 최종 스토리보드 및 기획 문서 (PDF)
